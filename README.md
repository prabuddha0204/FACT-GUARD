

# FactGuard 🛡️

### AI-Powered Misinformation & Scam Detection Platform

FactGuard is an AI-powered platform designed to help users detect **misinformation and digital scams** commonly spread through messaging platforms such as WhatsApp, SMS, and social media.

The system analyzes suspicious messages using AI, detects scam patterns like **UPI fraud, KYC scams, and fake job alerts**, extracts suspicious entities, and provides users with a **risk score and explanation** to help them make safer decisions online.

---

# 🚨 Problem

Misinformation and scams are spreading rapidly across digital platforms in India.

Common fraud types include:

* **UPI payment scams**
* **Fake KYC verification messages**
* **Job offer scams**
* **Phishing links**
* **Fake government schemes**

Many users do not have an easy way to **verify suspicious messages or links**, making them vulnerable to financial loss and misinformation.

---

# 💡 Solution

FactGuard allows users to submit suspicious messages or links, which are analyzed using AI to detect potential scams and misinformation.

The platform:

1. Analyzes the message using **AI/NLP**
2. Detects scam patterns (UPI, KYC, job scams)
3. Extracts entities such as phone numbers, domains, and UPI IDs
4. Checks these entities against a **scam intelligence database**
5. Generates a **risk score**
6. Educates the user about why the message is suspicious

---

# ⚙️ Key Features

### 🔍 AI Message Analyzer

Analyzes suspicious messages and detects scam patterns using natural language processing.

### ⚠️ Scam Detection

Identifies common scam types including:

* UPI payment fraud
* KYC verification scams
* Fake job alerts
* Phishing messages

### 🧠 Entity Extraction

Extracts important information from messages such as:

* Phone numbers
* Email addresses
* UPI IDs
* Domains / links

### 📊 Risk Score System

Generates a risk score indicating the likelihood of a scam.

Example:

```
Risk Score: 82%
Type: KYC Scam
Confidence: High
```

### 📚 User Education

Explains **why a message is suspicious**, helping users learn how to identify scams.
